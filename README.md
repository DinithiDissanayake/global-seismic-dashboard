# Global Seismic Activity Dashboard 
 
Interactive Streamlit dashboard analysing global earthquake activity 2020-2024. 
 
## Features 
- Interactive world map of earthquake locations 
- Filter by year, magnitude, depth, region and event type 
- Activity heatmap, trend charts and key insights 
- Download filtered data as CSV 
 
## Data Source 
USGS Earthquake Hazards Program via HDX 
 
## How to Run 
pip install -r requirements.txt 
streamlit run app.py
